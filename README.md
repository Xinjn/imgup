## Introduction

图片上传平台是⽤于快速⽣成在线图⽚链接的图床⼯具，简单⾼效。⽤⼾可以定制图⽚⼤⼩、浏览已上传的图⽚。开发了首页、鉴权、历史、用户、关于等6个页面，实现了登录注册、图⽚上传、图⽚定制、发布历史展⽰等功能。
